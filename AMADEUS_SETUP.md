# Amadeus API Setup Instructions

## Step-by-Step Signup

### 1. Go to Amadeus for Developers
**URL:** https://developers.amadeus.com

### 2. Click "Register" (top right)

### 3. Fill in the form:
- **Email:** dotandb@gmail.com (your email)
- **Password:** Create a strong password
- **First Name:** [Your first name]
- **Last Name:** [Your last name]
- **Company:** FlightPath Solutions
- **Country:** United Kingdom
- **Job Title:** Founder
- **Industry:** Travel

### 4. Verify your email
- Check inbox for verification link
- Click to activate

### 5. Create an App
- Go to "My Apps" in the dashboard
- Click "Create new app"
- **App Name:** FlightPath Search
- **Environment:** Production (or Self-Service)

### 6. Get Your Keys
Once created, you'll see:
- **API Key:** (long string, starts with random letters)
- **API Secret:** (another long string)

### 7. Copy and send to me
Paste both keys here and I'll integrate them.
